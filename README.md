# App-Development-Netflix-
# Netflix Clone

A Netflix-style web application built using **HTML, CSS, JavaScript, and Node.js**.
This project replicates the basic Netflix interface where users can browse movies, search content, and manage a personal watch list.

---

## 📌 Project Description

This Netflix Clone demonstrates full-stack web development concepts by combining a **Node.js backend** with a **dynamic frontend interface**.
The application loads movie data from the **TMDB API** and displays it in a Netflix-like layout with horizontal scrolling movie rows.

---

## 🚀 Features

* Netflix-style landing page
* User **Login and Signup**
* **Profile selection** screen
* Browse **Trending movies**
* **Search movies**
* Save movies to **My List**
* **Admin panel** to add movies
* Responsive Netflix-style interface
* Dynamic movie loading using **TMDB API**

## ⚙ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/310623104067/App-Development-Netflix-.git
```

### 2️⃣ Navigate into the project folder

```
cd App-Development-Netflix-
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Start the server

```
node server.js
```

### 5️⃣ Open the application

```
http://localhost:3000
```
